# 🏃‍♂️ Aplikasi Catat Lari

Aplikasi berbasis Android untuk membantu pengguna mencatat aktivitas lari mereka. Aplikasi ini dilengkapi dengan fitur registrasi pengguna, beranda informasi, profil, dan form pencatatan progres lari.

## 👥 Pembagian Tugas & Struktur Fragment

Pengembangan fitur dibagi menjadi beberapa *Fragment*. Berikut adalah daftar kontributor beserta tanggung jawab pembuatan Fragment (File *Logic* dan *Layout*):

* **Welcome dan Beranda**
    * **Kontributor:** Gratia Novelin Tamba (178)
    * **Nama File Logic:** `WelcomeFragment.kt` & `HomeFragment.kt`
    * **Nama File Layout:** `fragment_welcome.xml` & `fragment_home.xml`

* **Registrasi Akun**
    * **Kontributor:** Lutfia Nur Sabrina (175)
    * **Nama File Logic:** `RegisterFragment.kt`
    * **Nama File Layout:** `fragment_register.xml`

* **Profil**
    * **Kontributor:** Cutrin Joy M.T Sihombing (170)
    * **Nama File Logic:** `ProfileFragment.kt`
    * **Nama File Layout:** `fragment_profile.xml`

* **Form Pencatat Lari**
    * **Kontributor:** Kaka Dimas Soehendra Putra (171)
    * **Nama File Logic:** `RunFormFragment.kt` (atau `TrackerFragment.kt`)
    * **Nama File Layout:** `fragment_run_form.xml` (atau `fragment_tracker.xml`)

## 🛠️ Teknologi yang Digunakan
* **Platform:** Android
* **Bahasa Pemrograman:** Kotlin
* **Desain Antarmuka:** XML Layouts
